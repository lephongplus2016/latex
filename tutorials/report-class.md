## Cấu trúc của class report

- Cấu trúc:

  ```tex
  \documentclass{report}

  \begin{document}

  \chapter{Chapter}
  \section{Section}
  \subsection{Subsection}
  \subsection{Subsubsection}

  \end{document}
  ```
