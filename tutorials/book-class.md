## Cấu trúc của class book

- Cấu trúc:

  ```tex
  \documentclass{book}

  \begin{document}

  \part{Part}
  \chapter{Chapter}
  \section{Section}
  \subsection{Subsection}
  \subsection{Subsubsection}

  \end{document}
  ```
